## Installation
```
$ git clone https://github.com/PlayMaker-S/Sports-IT-Next.git

$ cd Sports-IT-Next
```

```
$ nvm use v.18.16.0

$ npm install

$ npm run dev
```

## Deploy
```
$ git clone https://github.com/PlayMaker-S/Sports-IT-Next.git

$ cd Sports-IT-Next
```

